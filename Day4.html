Basic Audio & Video Tags

Basic Implementation:

<audio src="./song.mp3" controls></audio>
<video src="./video.mp4" height="200" controls></video>

Attributes Explanation:

Tag/Attribute Description
<audio> / <video> Used to play sound and video files.
controls Adds play, pause, and volume buttons.
src Path of the file (local or online).
height / width Controls the media size.


Best Practice - Using <source> Tag

Recommended Implementation:

<audio controls>
  <source src="./song.mp3" type="audio/mp3">
</audio>
<video controls height="200">
  <source src="./video.mp4" type="video/mp4">
</video>

Why Use <source> Tag?

Reason Explanation
Multiple Formats You can add multiple formats (mp3, ogg, wav for audio / mp4, webm for video).
Better Compatibility Browser picks the format it supports â†’ better compatibility.


4. Embedding YouTube Videos

YouTube Embed Code:

<iframe src="https://www.youtube.com/embed/NGYp1zCBthk"></iframe>
<iframe src="https://youtube.com/embed/NGYp1zCBthk?si=BxL_jZO6xbBEtfRv" allowfullscreen></iframe>

iframe Attributes:

Attribute Description
<iframe> Used to embed content from another website (like YouTube, Google Maps).
allowfullscreen Lets users watch in full screen.


5. Multipage Website (Navigation)

Navigation Structure:

<header>
  <h1>This is our home Page</h1>
  <ul>
    <li><a href="index.html">HomePage</a></li>
    <li><a href="contact.html">Contact Us</a></li>
    <li><a href="course.html">Course</a></li>
    <li><a href="about.html">About us</a></li>
  </ul>
</header>

Navigation Elements:

Element Purpose
<header> Top section (title, navigation menu).
<ul> Unordered list (bullet points).
<li> Each menu item.
<a href="..."> Link to another page (index.html, about.html, etc.). This creates a multi-page website.


6. Main Content

Main Content Structure:

<main>
  <h2>This is our course selling website</h2>
  <img src="..." alt="course image">
</main>

Main Content Elements:

Element Description
<main> Holds the main content of the page.
<h2> Sub-heading.
<img> Displays an image.
src="data:image/jpeg;base64,..." Example of an inline image (encoded inside HTML instead of separate file).
